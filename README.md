# 学习指导 
* 学习要有节奏和保持良好的学习习惯，良好的作息安排和规律的生活习惯是管理精力和学习高效的必要条件；
* 采用先对大体的知识图进行了解，缩减学习内容。找到重点内容，理清这门课的解题逻辑，着重学习重点知识；  
* 在学习的过程中，一定要将知识点链接起来，不要去孤立的看待问题，对所学知识要进行拆解，逐步击破，一定要知道自己学的怎么样了；
* 不能一直处于一种似懂非懂，含糊不清的状态之中。要将感 性知识直接转化为熟练知识，非感性知识从完全不懂转换到，不熟练能做出来，再转化到对其进行细致的拆分，挨个击破其子知识点，使用多组训练的方式使其达到熟练程度；  
> 伤其十指不如断其一指。
*  在学习的过程中要注意采用正确的学习过程和循环，不要过于注重于对速度的追求，而是要明白自己的知识掌握程度；
*  再学习过程中一定要有反馈，包括正向激励的反馈和学习情况反馈，一旦处于那种自我感动和毫无激励的情况纸中就会陷入痛苦和怀疑或者是盲目的自大之中；
*  要相信世界上没有所谓的不会的知识点，有的只是自己逻辑链断裂的知识点，这样在面对高层次的抽象时，会使自己无法理解一个别人认为很简单的问题，进而导致自己的信心受挫，而我们要做的找到自己缺失的那条逻辑链，补全就可以了；
*  要做到痴迷而不是强迫自己混日子，要自己一直去想着问题，而不是坐在那里一到时间就什么都不想了，学习的时候又想着其他的事情；
*  如果遇到不会的题目自己想了一段时间后确实想不出来，就直接看答案，去想一想为什么这么写，别人的思路是怎么来的，自己有没有完全弄明白题目的意思、知道所有的条件、有没有对条件和题目做相应的变动、有没有想办法用自己的话复述一遍问题，这个题目考了那些知识点，这些考法有没有什么新颖之处；
*  分数是过程的结果，一个正确可循环的过程能够给学习带来巨大的推动力；

# 对学习的改进
* 对于学习要有一个好的流程，不可以没有目标，但也不能完全以完成任务为目的，而是有一个整体的战略目标，在此基础上对其实现来进行拆分；
## 我的目标是考上一个考408的学校的研究生：
### 树立起正确的学习观念
* 放下畏难情绪，先自己做了看看，不行就想办法请教别人；
* 一旦想好学习就不要想着先玩一会在学习，而是想着先学学完今天的进度可以奖励一下，或者这一天快要结束了稍微娱乐一会（一定要注意度量）；
* 要知其然还要知其所以然，要想着如何想出答案，考点是什么，用出题人思维来解决问题；
* 要明白复利的强大，每天哪怕只是一点点，但在长时间的积累就会有巨大的收获；
### 适合自己的可循环的学习流程
* 学习是一个完整的流程，如果只是重复其中一个步骤那是没有办法真正学到知识的；
1. 明确知识点：在学习的过程中一定要明白自己学的东西是什么，可以将知识点大致分为**可泛化**和**记忆类**两种；
2. 寻找专业教材：专业教材中具有专家对这个知识点的严谨的描述，可以帮助我们大致的建立起一个框架；
3. 寻找对应的例子：通过对一定量的例子进行观察和归纳，使自己可以对照着**指令**建立起大致的知识点（模型）；
4. 做对应的练习题：验证自己所构建的模型是否正确，检查自己在解题过程中的熟练度，通过对照答案（包括过程）来总结自己的模型是否有问题或偏差；
5. 和之前做过的题目进行比较：寻找异同判明知识的使用范围（真理的有界性），并进行归纳；
6. 新题验证（反馈）：看自己的修正过的模型是否有效，若无效则返回2； 
* 对应的一个更加通用的流程：
1. 先看一下学习指导，明确自己要学的知识有那些做一个列表；
2. 先大致浏览一遍书，不追求看懂，要想把书看懂要花费过多的时间；
3. 看对应的课程视频，对知识点进行重述和标记重点；
4. 自己在对照的例子进行理解和找更多的例子；
5. 浏览一遍所学的知识点进行习题练习；
6. 对答案订正，根据答案进行修正模型，比对类似题型（一定是在完成一定量的题目后再订正）；
7. 做新的题目验证；
8. 制作知识点卡片，并在学习完成一章后制作思维导图（一定要完成不断迭代更新和删除已掌握知识点），并且将知识点的熟练程度进行分类和标记，对于熟练的知识点可以不管但是不熟练的知识点，要对其进行拆分并且要以多组进行专项训练；
